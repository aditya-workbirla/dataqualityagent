Executive Summary

The dataset covers Q3 2024 production runs across the Kraft digester process. Overall data quality scores 74/100 — acceptable but with two critical issues that need immediate attention before downstream modeling.

Critical: Negative Temperature Values in Stage 4

Based on research of the Kraft pulping process, digester temperatures must remain between 155°C and 175°C at all stages. The dataset contains 34 rows where digester_temp_stage4 falls below 0°C. These are physically impossible readings and indicate likely sensor calibration failures or data pipeline errors. These rows should be flagged and excluded from analysis.

Critical: pH vs. Chemical Flow Logical Inconsistency

A domain-specific check (new function generated) cross-referenced pH readings against caustic_flow_rate. In Kraft processing, high caustic flow should correlate with elevated pH. 89 rows show high caustic flow (>45 L/min) with pH below 9.5, which is a logical impossibility in standard Kraft chemistry and likely represents flow meter misreads.

Warnings: Missing Values in Pressure Columns

press_vessel_2 has a 7.8% null rate, which is unusually high. For continuous process monitoring, gaps of this length may correspond to scheduled maintenance periods — recommend validating against the maintenance log.
